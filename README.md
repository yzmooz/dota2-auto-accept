
# Dota 2 Auto Accept

Автоматически нажимает кнопку **«ПРИНЯТЬ»**, когда в Dota 2 находится матч.

Скрипт ждёт **настоящие события Windows**:

1. **`HSHELL_FLASH`** — окно Dota замигало в панели задач (то самое красное
   мигание при найденном матче) — главный триггер;
2. **`EVENT_SYSTEM_FOREGROUND`** — окно Dota вышло на передний план — запасной
   триггер.

Только в момент срабатывания триггера делается **один** поиск кнопки через
OpenCV (`matchTemplate` по эталону `images/accept_button.png`) и клик по ней.

## Установка

Нужен **Python 3.10+** и **Windows** (используются WinAPI-хуки).

```bash
git clone https://github.com/yzmooz/dota2-auto-accept.git
cd dota2-auto-accept


python -m venv .venv
source .venv/Scripts/activate      # PowerShell: .venv\Scripts\Activate.ps1

pip install -r requirements.txt
```

## Использование

Проверить, видит ли скрипт кнопку прямо сейчас (открой ready-check или покажи
скриншот кнопки на экране):

```bash
python accept_dota.py --test          # покажет уверенность, не кликая
python accept_dota.py --test --click  # то же + реальный клик (для проверки)
```

Рабочий режим — запусти перед поиском игры и оставь висеть:

```bash
python accept_dota.py
```

Остановка: **Ctrl+C** в консоли. (Увод курсора в левый верхний угол экрана —
это аварийная отмена клика pyautogui, а не выключение скрипта.)

## Свой эталон кнопки — `make_template.py`

Готовый эталон `images/accept_button.png` снят на **1920×1080, русский
интерфейс**. Поиск мультимасштабный и переживает небольшую разницу, но при
другом разрешении или языке интерфейса лучше пересоздать эталон под себя.
`make_template.py` сам находит зелёную кнопку «ПРИНЯТЬ» на твоём скриншоте и
вырезает её в `images/accept_button.png`.

```bash
# 1. Сделай скриншот всего экрана с видимой кнопкой
#    (Win+PrintScreen сохраняет в «Изображения/Снимки экрана»).
# 2. Сгенерируй эталон — зелёная кнопка находится автоматически:
python make_template.py "C:/путь/к/скриншоту.png"
```

Если автопоиск не сработал (кнопка другого цвета), задай область вручную в
пикселях `X Y W H`:

```bash
python make_template.py "C:/путь/к/скриншоту.png"  X Y W H
```

## Настройки

В начале файла `accept_dota.py`:

| Параметр           | Назначение                                                            |
|--------------------|-----------------------------------------------------------------------|
| `MATCH_THRESHOLD`  | Порог совпадения 0..1. Ложно кликает → повысь; не находит → понизь.   |
| `DEBOUNCE_SECONDS` | Пауза после клика, чтобы не нажимать повторно.                        |
| `SAFETY_SCAN_SEC`  | Фоновый страховочный скан раз в N сек (`0` = чисто событийный режим). |
| `RETRY_SECONDS`    | Сколько искать кнопку после триггера.                                 |

## Дисклеймер

Скрипт делает обычный клик по экрану и **не вмешивается в процесс игры** (не
читает и не меняет память Dota). Тем не менее используешь на свой риск.
